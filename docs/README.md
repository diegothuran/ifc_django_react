## 📦 Principais Recursos

### 🏗️ Visualizador IFC
- Renderização 3D com Three.js
- Seleção e inspeção de elementos
- OrbitControls profissionais
- Cache de metadados (7 dias)

### 🔌 API REST
- 13+ endpoints
- Metadados IFC completos
- Busca de elementos
- Estatísticas do modelo

### 📊 Backend
- Django 5.2.7
- IfcOpenShell 0.8+
- PostgreSQL cache
- WhiteNoise static files
